lua53-u8w
======

Lua 5.3.5 with UTF-8 Wrapper for Windows

Converting from UTF-8 to UTF-16(LE) and vice versa at Windows API and C runtime functions.

See lu8w.c and lu8w.h source files, and U8W_H macro.
